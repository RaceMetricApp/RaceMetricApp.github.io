# RaceMetric Website

This folder contains the free static website for RaceMetric.

Recommended no-cost hosting:

1. Create a public GitHub repository under the `RaceMetricApp` organisation named `RaceMetricApp.github.io`.
2. Upload the files from this `website` folder into the root of that repository.
3. Open the repository settings, then Pages, and set the source to the main branch.
4. After GitHub publishes it, the site should be available at `https://racemetricapp.github.io/`.

App Store Connect URLs:

- Marketing URL: `https://racemetricapp.github.io/`
- Privacy Policy URL: `https://racemetricapp.github.io/privacy.html`
- Support URL: `https://racemetricapp.github.io/support.html`
